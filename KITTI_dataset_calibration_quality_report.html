Understanding the Metrics

This report assesses the alignment between Camera and LiDAR sensors using the following metrics:

  • Geometric Edge Score (0.0 - 1.0): Measures how well edges detected in the camera image align with depth discontinuities (edges) in the projected LiDAR point cloud.
    Interpretation: Higher is better. Values > 0.10 typically indicate reasonable alignment for complex scenes. Values near 0.0 indicate misalignment or featureless scenes.
  • Mutual Information (NMI): Measures the statistical dependence between the camera image intensity and the LiDAR return intensity.
    Interpretation: Higher is better. Values > 0.05 typically indicate that the sensors are observing the same scene structure. Values near 0.0 suggest independence (misalignment).
  • Contrastive Score: A learned similarity metric using deep neural networks (if enabled).
